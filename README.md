# Smart-Response-And-Monitoring-Rover

This mini project, developed by Aiswarya N L,Arjun N K,Shilpa Ann Varghese and me, focuses on an Intelligent Robotics and Embedded System designed for disaster response and surveillance. The system integrates sensors, cameras, and wireless communication technologies to provide real-time monitoring, hazard detection, and emergency alerts in critical environments.

The project aims to reduce human risk in dangerous zones by enabling early detection of victims, obstacles, and potential hazards. By supporting faster and safer rescue operations, this system demonstrates the practical application of robotics and embedded systems in disaster management and public safety.
